  • Spreadsheet hyperlink feature does not work while editing cell content
  • Change event is not triggered on row deletion
  • Date parsing does not take current culture into account
  • JS error when bound to datasource with date field
  • Border tools navigate to # when clicked
  • Clicking outside widget keeps cell editor open
  • JS error when attempting to submit value for a disabled cell and then navigate to next cell with Tab
  • JS error on deleting a sheet while renaming another sheet
  • Filtering by value (Blanks) is not applied when clicking Apply
  • Edit command undo action reverts cell to blank if its previous value was 0
  • Inserting and deleting rows and columns doesn't update their width and height
  • Row / column resizing cannot be undone
